PerFly is the new performance inspector for the Android platform

PerFly allows the investigation of key values such as memory footprint, CPU usage and other key events of the Android/Dalvik platform, such as Garbage Collection, application status, interface orientation.

Complete profiling

PerFly allows you to inspect the following values and events:

Memory

  • total memory footprint
  • native memory footprint of the application
  • Dalvik memory footprint of the application
  • memory limit of the system’s Dalvik machine

CPU

  • total CPU consumption
  • user’s CPU consumption
  • kernel’s CPU consumption

Application events

  • state
  • PID

System events

  • orientation

Garbage-Collection events (requires root access)

  • GC_Reason
  • Amount_freed
  • Heap_stats
  • External_memory_status
